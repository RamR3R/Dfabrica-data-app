
# DFabrica API

This is the API repo used for the Deployment of the API data of the Assests of DFabrica Website.


## API Reference

#### Get all items

```http
  GET http://dfabrica-data-app.onrender.com/users
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | gets all the user data |

#### Get item

```http
  GET http://dfabrica-data-app.onrender.com/users/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of user to fetch |
